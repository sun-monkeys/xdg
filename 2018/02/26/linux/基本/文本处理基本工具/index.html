文件查看　cat

分页查看文件内容　more　less

显示文本前或后行内容　head　tail

按列抽取文本和合并文件　cut　paste

显示文件或STDIN数据的指定列

合并两个文件同行号的列到一行paste

收集文本统计数据　wc

  • 计数单词总数、行总数、字节总数和字符总数
  • 可以对文件或STDIN中的数据运行

文本排序　sort

把整理过的文本显示在STDOUT，不改变原始文件

从输入中删除前后相接的重复的行　uniq

比较文件　diff　patch

2、查出分区空间使用率的最大百分比值

3、查出用户UID最大值的用户名、UID及shell类型

4、查出/tmp的权限，以数字方式显示
